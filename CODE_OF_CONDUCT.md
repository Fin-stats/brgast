# Code of Conduct

## Our pledge

We want participation in the `brgast` project to be respectful, constructive,
and welcoming for everyone.

## Expected behavior

- Be respectful in language and tone.
- Assume good intent and discuss disagreements constructively.
- Focus feedback on ideas, code, and documentation.
- Help keep the project useful for the broader R community.

## Unacceptable behavior

- Harassment, insults, or personal attacks
- Bad-faith disruption of issues or pull requests
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, or
other contributions that do not align with this code of conduct.
